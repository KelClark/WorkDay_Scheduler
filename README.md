# Work Day Scheduler

## Description
This Work Schedule Planner is a simple and efficient web application that allows users to schedule their work hours effectively. It provides an easy-to-use interface where users can add and save events for each hour of the workday.
## Features: 
- Current day is displayed at the top of the calendar for easy reference.
- Time blocks are color-coded to indicate whether they are in the past, present, or future.
- Users can click into a time block to enter an event.
- Events are saved to local storage, allowing them to persist even when the page is refreshed.
## Usage
1. Open the Work Schedule Planner.
2. Scroll down to view the time blocks for the day.
3. Click into a time block to enter an event.
4. Click the save button for that time block to save the event.
5. If you refresh the page, your saved events will still be there.
